
## 2️⃣ 단순화: 8살 아이도 이해할 수 있게

> **핵심**: 수식 없이, 비유로 설명하기. 이것이 진짜 내가 이해했는지 확인하는 척도.

### 가장 단순한 설명
[개념]은 **[비유1]**과 같다.

예를 들어:
```
❌ Attention은 Query, Key, Value를 통해 weighted sum을 계산한다
✅ Attention은 도서관에서 책을 찾는 과정이다
   - 내가 찾는 책 (Query: "AI에 관한 책")
   - 도서관의 모든 책 (Keys: 책의 제목, 저자, 주제)
   - 도서관의 모든 책 내용 (Values: 책의 내용)
   → 내 질문과 가장 관련 있는 책들을 찾아서 관련 내용을 모은다
```

### 일상 속 유사한 예
[개념]과 유사한 일상의 예를 찾기:
- 예시 1: [일상적인 상황]
- 예시 2: [또 다른 상황]

이렇게 비유하니 이해가 쉬워진다:
- [이해한 부분 1]
- [이해한 부분 2]
